FAQ

Our AI system uses a combination of Deep Q-Learning and Graph Neural Networks to analyze traffic patterns and make real-time decisions about potential threats. The system continuously learns from new attack patterns to improve detection accuracy.

dDOS.Ai can detect and mitigate various types of DDoS attacks, including volumetric attacks (UDP floods, ICMP floods), protocol attacks (SYN floods, fragmented packet attacks), and application layer attacks (HTTP floods, slow loris, DNS query floods).

Our system typically responds to attacks within milliseconds, with continuous monitoring and real-time mitigation capabilities. The average response time is 45ms, but this can vary based on attack complexity and network conditions.
